# 🚀 数据建模课程环境一键配置指南

## 💻 使用前准备

1. 电脑需要连接网络。
2. 下载的课程压缩包[链接](https://github.com/zhoujing204/data-modeling-course-student/archive/refs/heads/master.zip)并解压，请勿直接在压缩软件内双击运行！

---

## 🏃‍♂️ Windows安装脚本运行步骤

### 步骤 1：定位到课程文件夹

打开您解压好的课程主目录（例如 `DATA-MODELING-COURSE-STUDENT`）。请确保在这个目录下能看到 `requirements.txt` 文件和我们的启动脚本文件（`.bat` 格式）。

### 步骤 2：运行一键脚本

找到名为 `Windows一键安装脚本.bat`，双击运行它。

### 步骤 3：等待自动化安装

运行后，会弹出一个黑色的命令行窗口。脚本将自动按顺序执行所有步骤：

* 当安装 Python 和 VS Code 时，**屏幕上会弹出带有进度条的安装窗口**。
* **⚠️ 重要提示：** 请耐心等待进度条走完，**绝对不要手动关闭弹出的安装窗口或黑色的命令行窗口！** 脚本会在每个步骤完成后自动继续。
* 整个过程大约需要 15 分钟，具体取决于您的网络和电脑速度。

### 步骤 4：安装完成

当命令行窗口提示 **`环境配置全部完成！`** 并显示“按任意键继续...”时，恭喜您，所有的环境都已经准备就绪了！您可以按任意键关闭该窗口。

## 🏃‍♂️ 苹果电脑MacOS安装脚本运行步骤

### 步骤 1：定位到课程文件夹

打开您解压好的课程主目录（例如 `DATA-MODELING-COURSE-STUDENT`）。请确保在这个目录下能看到 `requirements.txt` 文件和我们的启动脚本文件（`.sh` 格式）。

### 步骤 2：运行一键脚本

由于 macOS 系统的安全机制，无法直接双击运行脚本，请按以下步骤操作：

1. 打开 Mac 系统自带的 **终端 (Terminal)** 应用程序（可以在“启动台”的“其他”文件夹中找到，或通过聚焦搜索搜索“终端”）。

2. 在终端中输入 `cd`（**注意 `cd` 后面必须有一个空格**），然后将您刚才解压好的 **课程主目录文件夹** 直接拖拽到终端窗口中，按下回车键。

3.赋予脚本执行权限：

```bash
chmod +x macOS一键安装脚本.sh
```

4. 接着，在终端中输入命令 `sh macOS一键安装脚本.sh`（请替换为您实际保存的脚本名称），按下回车键即可开始运行。

### 步骤 3：等待自动化安装

运行后，终端窗口会自动按顺序执行所有步骤下载并配置环境：

* 当安装 Python 官方包或配置 VS Code 权限时，**终端会提示输入您的 Mac 登录/开机密码**。**(注意：输入密码时屏幕上不会显示任何星号或字符，这是正常的，输入完毕后直接按回车即可)**。
* **⚠️ 重要提示：** 请耐心等待终端中的代码滚动执行，**绝对不要手动关闭终端窗口或强行退出！** 脚本会在每个步骤完成后自动继续。
* 整个过程大约需要 15 分钟，具体取决于您的网络和电脑速度。

### 步骤 4：安装完成

当终端窗口输出绿色的 **`环境配置全部完成！`** 提示语时，恭喜您，所有的环境都已经准备就绪了！您可以直接关闭终端窗口，并在 Mac 的“启动台 (Launchpad)”或“应用程序”文件夹中找到 Visual Studio Code 开始学习。
